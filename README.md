# Yogana Portfolio

A personal portfolio built with React, TypeScript, and Vite.

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS *(if you're using it — remove if not)*

## Getting Started

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

## Environment Variables

Create a `.env` file based on `.env.example`:

```bash
cp .env.example .env
```

## Project Structure

```
src/
  components/
  pages/
  assets/
  hooks/
  services/
```

## Notes

* Environment variables must start with `VITE_`
* `.env` is ignored from version control
